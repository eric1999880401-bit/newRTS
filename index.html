Method used in this app
e1RM is calculated from load / %1RM. The %1RM is taken from an RTS-style reps x RPE table. Back-off recommendations multiply the selected e1RM by the target table percentage, then round to the user's chosen plate increment. Fatigue drop is computed as: (reference e1RM - current e1RM) / reference e1RM x 100.
